## What I'm working on

My thesis focuses on integrating environmental metrics (energy, CO₂e) directly into MLOps observability pipelines, so teams can treat sustainability as an operational signal rather than something measured after the fact. The practical side of that work lives here:

**[huggingface-etl-co2emissions](https://github.com/jufegare000/huggingface-etl-co2emissions)** — An ETL pipeline on AWS Glue + PySpark that collects and consolidates CO₂ emissions metadata from Hugging Face models. Bronze → Silver → Gold medallion architecture, fault-tolerant, with Terraform for infrastructure. It's the data foundation for analyzing how the open-source ML ecosystem actually reports its environmental footprint.
